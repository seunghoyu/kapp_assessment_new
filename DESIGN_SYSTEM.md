# 디자인 시스템 요약

## 컬러 토큰 정의

### 포인트 컬러 (3가지 - 보조적 UI 요소에만 사용)

#### 1. Blue 계열 (Primary - 브랜드/캠퍼스 포인트/텍스트 강조)
- **용도**: 메인 포인트 컬러
- **사용 예시**:
  - 활성 상태 표시 (선택된 메뉴, 활성 탭)
  - 링크/버튼 (Primary 버튼)
  - 중요 정보 강조
  - KPI 카드의 포인트 라벨
  - 차트 포인트/라인
- **토큰명**: `blue-*`
- **색상 값**: 
  - `blue-50`: 매우 연한 배경
  - `blue-100`: 연한 배경
  - `blue-500`: 기본 포인트 (버튼, 링크)
  - `blue-600`: 호버 상태
  - `blue-700`: 활성/강조 상태
  - `blue-900`: 텍스트 강조

#### 2. Yellow 계열 (Accent - 카테고리/상태/주의 포인트)
- **용도**: 주의/경고/카테고리 표시
- **사용 예시**:
  - 경고 배지/태그
  - 진행 중 상태
  - 카테고리 구분
  - 중요도 중간 표시
- **토큰명**: `yellow-*`
- **색상 값**:
  - `yellow-50`: 매우 연한 배경
  - `yellow-100`: 연한 배경
  - `yellow-500`: 기본 포인트
  - `yellow-600`: 호버 상태
  - `yellow-700`: 강조 상태

#### 3. Purple 계열 (Highlight - 특정 강조 섹션/배너/하이라이트)
- **용도**: 특별한 강조/프리미엄 기능
- **사용 예시**:
  - AI 인사이트 배지
  - 프리미엄 기능 표시
  - 특별 이벤트/배너
  - 하이라이트 섹션
- **토큰명**: `purple-*`
- **색상 값**:
  - `purple-50`: 매우 연한 배경
  - `purple-100`: 연한 배경
  - `purple-500`: 기본 포인트
  - `purple-600`: 호버 상태
  - `purple-700`: 강조 상태

### Gray Scale (기본 UI 톤)

#### 기본 배경/텍스트 컬러
- **배경**: `white` / `gray-50` (very light gray)
- **카드**: `white`
- **구분선/보더**: `gray-200` (light gray)
- **사이드바/헤더**: `white` 또는 `gray-50`
- **텍스트**:
  - `gray-900`: 주요 텍스트 (제목, 본문 강조)
  - `gray-700`: 일반 텍스트
  - `gray-500`: 보조 텍스트 (서브타이틀, 힌트)
  - `gray-400`: 비활성 텍스트

## 컬러 사용 규칙

### ❌ 사용 금지 영역
- **전체 배경**: 포인트 컬러를 배경 전체에 사용하지 않음
- **사이드바 배경**: 포인트 컬러 사용 금지 (white/gray-50만 사용)
- **헤더 배경**: 포인트 컬러 사용 금지 (white/gray-50만 사용)
- **메인 콘텐츠 배경**: 포인트 컬러 사용 금지 (white/gray-50만 사용)
- **카드 배경**: 포인트 컬러 사용 금지 (white만 사용)

### ✅ 사용 가능 영역 (보조적 UI 요소)
1. **태그/배지/칩**
   - 테이블 내 상태 태그
   - 카드 내 카테고리 배지
   - 필터 칩

2. **상태 표시**
   - 활성/비활성 상태
   - 진행 중/완료 상태
   - 중요도 표시

3. **선택된 항목 강조**
   - 선택된 메뉴 아이템
   - 선택된 탭
   - 선택된 필터

4. **KPI 카드 포인트**
   - 카드 내 작은 포인트 라벨
   - 증감률 표시 (델타)

5. **차트 요소**
   - 범례 색상
   - 포인트 마커
   - 하이라이트 라인

6. **버튼/링크**
   - Primary 버튼 (blue)
   - 중요 액션 버튼
   - 링크 텍스트 (blue)

## 아이콘 시스템

### 원칙
- **이모지 사용 금지**: 모든 이모지 제거
- **통일된 스타일**: 하나의 아이콘 라이브러리 사용 (Lucide React)
- **의미 전달 보조**: 아이콘은 텍스트와 함께 사용, 단독 사용 지양
- **과도한 사용 지양**: 필요한 곳에만 사용

### 아이콘 라이브러리
- **Lucide React**: 통일된 아이콘 세트
- **크기**: 
  - Sidebar: 20px (w-5 h-5)
  - 버튼/카드: 16px (w-4 h-4)
  - 작은 배지: 14px (w-3.5 h-3.5)

### 아이콘 사용 위치
- Sidebar 메뉴 아이템
- 버튼 내부 (액션 아이콘)
- 카드 헤더 (카테고리 아이콘)
- 상태 표시 (성공/경고/에러)
- 테이블 액션 (편집/삭제 등)

- **그림자**: `shadow-sm`
- **제목**: `text-gray-900`

## 결과적 톤

**"차분한 대시보드 + 포인트 컬러로만 의미 강조"**

- 전체적으로 화이트/라이트 그레이 기반의 깔끔한 톤
- 포인트 컬러는 작은 UI 요소(태그, 배지, 상태, 선택)에만 제한적으로 사용
- 기업용 대시보드의 전문적이고 차분한 느낌 유지
- 가독성과 일관성 우선
