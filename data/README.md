# data folder structure

Data is grouped by feature. Use `@/data/<folder>/<file>` for imports.

- **dashboard** - Dashboard overview: strategyMapper.json, departmentPerformance.ts, departmentSkillPerformance.ts
- **competency** - Competency analysis: competencyRawData.ts, highPerformerData.ts
- **benchmark** - Industry benchmarking: organizationData.json, industryBenchmarks.json, loadBenchmarkData.ts, transformBenchmark.ts
- **employees** - Employee/HR: employees.ts, employeesData.ts
- **kappDiagnosis** - KAPP diagnosis reference: userInfoOptions.json, assessmentResultSample.json
- **_legacy** - Currently unused JSON (for reference or future API)
