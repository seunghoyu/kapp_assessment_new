# kapp_assessment_new – 마케터 관점 프로젝트 개요

> **역할**: 타깃, 메시지, 차별화, 지표, CTA·브랜딩  
> **초점**: new 서비스의 레이아웃·간소화·키 컬러 등 변경점을 메시징에 반영

---

## 1. 타깃 (new 기준)

### 1.1 1차: 기업·공공의 교육담당·HR
- **역할**: 조직 역량·교육 현황 파악, 의사결정
- **니즈**: 한 화면에 요약·부서 비교·벤치마킹
- **new에서의 메시지**: “사이드바로 한 번에 이동·간단한 KPI·부서별 차트·업계 벤치마킹으로 데이터 기반 판단”

### 1.2 2차: 임직원(진단 응시자·학습자)
- **현재 new**: 해당 화면 없음. 추후 “진단·마이 대시보드·교육·나의 성장” 구현 시 타깃.

---

## 2. 핵심 메시지·포지셔닝 (new 반영)

### 2.1 한 줄 포지셔닝
**“Next.js 기반 관리자 대시보드로 조직 역량·벤치마킹을 한 화면에서, 간결한 레이아웃과 키 컬러로 핵심만 전달”**

### 2.2 서브 메시지
- **레이아웃**: 사이드바·헤더로 이동과 정보 탐색이 명확함.  
- **간소화**: 데이터 노출을 줄여 “요약 KPI·부서 차트·벤치마킹·역량 분석”에 집중.  
- **키 컬러**: blue(활성·강조), yellow(진행), purple(향상) 등으로 상태·강조만 구분.

---

## 3. 페이지별 메시징·CTA (구현된 부분 위주)

### 3.1 로그인(`/`)
- **메시지**: “계정 정보를 입력해 주세요”  
- **CTA**: 로그인 버튼 → (자리표시) 대시보드 이동

### 3.2 대시보드 개요(`/dashboard`)
- **헤더**: “대시보드” / “조직 분석 및 관리”  
- **블록**: 총 직원 수·진행 중 프로그램·완료율·평균 스킬 레벨(KPI 카드), 부서별 역량 현황·스킬 분포, 이번 달 요약·주요 성과·다음 액션, **역량 벤치마킹**(부서 vs 업계 상위 10%)  
- **CTA**: (현재 명시적 CTA는 적음. 추후 “리포트 다운로드”·“설정” 등)

### 3.3 역량 분석(`/dashboard/competency`)
- **헤더**: “역량 분석” / “부서·기간·역량 기준 통합 분석”  
- **탭**: 현황&비교 / 격차&리스크 / 전략&성과  
- **메시지**: 필터로 부서·기간 선택 시 “요약 KPI·부서별 막대·기간별 추이·벤치마킹”으로 한눈에 비교

### 3.4 미구현 페이지
- **예정 메시지/CTA**: 진단 시작 → 마이 대시보드, 맞춤 교육, 나의 성장·인증서 등은 origin 04_project_brief_marketer.md 참고하여, 구현 시 동일한 메시지·CTA 적용

---

## 4. 지표·전환 (관리자 대시보드)

- **전환 지표로 볼 수 있는 액션**:  
  - 필터 사용(부서·기간·년/월·일/월 추이)·탭 전환(현황/격차/전략)  
  - 벤치마킹에서 부서·업계 선택 변경  
  - (추후) 다운로드·설정·교육 배정·리포트 생성 등 버튼 클릭  
- **관리자 페이지 방문·기능 사용률**: 대시보드 진입, 개요 vs 역량 분석 이동, 필터·탭 사용 횟수로 “어떤 블록을 쓰는지” 파악 가능.

---

## 5. 브랜딩·키 컬러

- **new에서 적용 중**:  
  - 화이트·라이트 그레이 배경, 카드·테이블 border-gray-200.  
  - **포인트 컬러**: blue(활성 메뉴·강조), yellow(진행중·경고), purple(향상·특별 강조).  
- **마케팅 활용**:  
  - “한눈에 보는 상태 구분” — 활성/진행/향상을 색으로만 구분해 가독성·전문성 전달.  
  - 자료·스크린샷 시 “키 컬러만 강조된 깔끔한 대시보드”로 차별화 메시지.

---

## 6. 정리

- **타깃**: 현재는 **관리자(교육담당·HR)** 위주. 학습자·진단은 미구현.  
- **메시지**: 사이드바·간소화·키 컬러 기반의 “명확한 관리자 대시보드”.  
- **지표**: 필터·탭·벤치마크 선택 등 액션을 전환 지표로 활용.  
- **브랜딩**: 키 컬러를 상태·강조에만 사용해 전문적·차분한 인상 유지.
