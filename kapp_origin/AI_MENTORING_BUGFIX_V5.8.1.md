# AI 커리어 멘토링 강의 추천 버그 수정 (v5.8.1)

## 🐛 문제 상황

**증상**:
- ❌ AI 멘토에게 "어떤 교육을 들어야 하나요?" 질문 시 같은 말만 반복
- ❌ 강의 추천이 작동하지 않음
- ❌ 이전(v5.6)에는 정상 작동했으나 v5.7 교육 데이터 확장 후 문제 발생

**사용자 경험**:
```
사용자: "어떤 교육을 들어야 하나요?"

AI (잘못된 응답):
"데이터 사이언티스트를 위한 맞춤 강의를 찾고 있습니다.

교육 큐레이션 페이지에서 더 많은 강의를 확인해보세요!"
```

## 🔍 원인 분석

### 1. 데이터 필드명 불일치
**Before (v5.6)**: `education-data-extended.js` (28개 강의)
```javascript
{
    id: 1,
    title: "...",
    applicableIndustries: ["IT", "금융"] // 구 필드명
}
```

**After (v5.7)**: `education-data-comprehensive.js` (65개 강의)
```javascript
{
    id: 101,
    title: "...",
    industries: ["IT", "금융", "제조"] // 신 필드명
}
```

**문제 코드 (js/my-growth.js Line 658)**:
```javascript
recommendedCourses = recommendedCourses.filter(course => {
    return keywords.some(kw => 
        (course.applicableIndustries && ...) // ❌ undefined!
    );
});
```

→ `course.applicableIndustries`는 존재하지 않아 모든 강의가 필터링되어 결과 0개

---

### 2. 카테고리 매핑 불일치
**Before (v5.6)**: 6개 추상 카테고리
```javascript
const categoryMap = {
    'knowledge': ['학습', 'IT/개발/데이터', '자기계발'],
    'application': ['문제 해결', '비즈니스/전략', 'IT/개발/데이터'],
    'performance': ['협업', '리더십/조직', '의사소통/협업'],
    'productivity': ['기술', 'IT/개발/데이터', '자기계발']
};
```

**After (v5.7)**: 8개 실무 카테고리
```
디지털/IT, 비즈니스/경영, 마케팅/영업, 리더십/HR, 
커뮤니케이션, 자기계발, 직무 전문, 어학
```

**문제**:
- `'학습'` 카테고리는 존재하지 않음
- `'IT/개발/데이터'`는 `'디지털/IT'`로 변경됨
- `'비즈니스/전략'`은 `'비즈니스/경영'`으로 변경됨

→ 카테고리 매칭 실패로 강의 0개 반환
