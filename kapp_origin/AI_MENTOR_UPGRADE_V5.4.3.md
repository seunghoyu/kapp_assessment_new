## 🎯 문제 해결

### 🔴 사용자 요청
> "나의 성장 탭에서, AI 커리어 멘토 기능에서 어떤 교육을 들을지 물었을때, 아까 우리 해커스 캠퍼스 business 홈페이지 내에 있던 강의들 중에서 관련된 강의를 추천해주도록해. 지금은 어떤 교육을 들을지, 교육 커리큘럼을 추천해달라는 질문에도 똑같은 말만 반복하고 있어서 좀 그래"

### ✅ 해결 완료
**Before (문제 상황)**:
- ❌ AI 멘토가 **랜덤 응답 3가지만 반복**
- ❌ "어떤 교육을 들어야 하나요?" 질문에 **구체적인 강의 추천 없음**
- ❌ **해커스 캠퍼스 Business 실제 강의와 연동되지 않음**
- ❌ 똑같은 일반적인 답변만 반복

**After (해결 후)**:
- ✅ **질문 유형별 스마트 응답**: 교육/기간/전략/스킬 질문 자동 인식
- ✅ **해커스 캠퍼스 Business 28개 실제 강의 추천**: coursesData 연동
- ✅ **KAPP 점수 기반 맞춤 추천**: 약점 역량에 맞는 강의 선별
- ✅ **목표 직무별 필터링**: 마케팅/PM/데이터 사이언티스트 등 맞춤 추천
- ✅ **우선순위 표시**: 🔥 최우선, ⭐ 추천, 3. 등으로 순서 제시
- ✅ **상세 정보 제공**: 카테고리, 난이도, 기간, 평점, 수강생 수
- ✅ **학습 순서 제안**: 1→2→3 단계별 로드맵

## 🧠 AI 멘토 스마트 응답 시스템

### 질문 유형 자동 인식

| 질문 키워드 | 응답 유형 | 제공 정보 |
|------------|----------|----------|
| **교육, 강의, 수강, 커리큘럼, 학습** | 강의 추천 | 해커스 HRD 강의 5개 + 우선순위 |
| **기간, 시간, 달성, 언제, 얼마나** | 타임라인 | 6-15개월 예상 + 단계별 일정 |
| **어떻게, 방법, 전략, 순서, 먼저** | 전략 제시 | 우선순위 + 효과적인 학습 방법 |
| **스킬, 역량, 능력** | 스킬 분석 | 필요 스킬 목록 + 목표 점수 |

### 강의 추천 알고리즘

```javascript
1. KAPP 점수에서 약점 분석
   - Knowledge ↓ → '학습', 'IT/개발/데이터', '자기계발'
   - Application ↓ → '문제 해결', '비즈니스/전략', 'IT/개발/데이터'
   - Performance ↓ → '협업', '리더십/조직', '의사소통/협업'
   - Productivity ↓ → '기술', 'IT/개발/데이터', '자기계발'

2. 목표 직무별 키워드 매핑
   - 마케팅 팀장 → 마케팅, 광고, 비즈니스, 전략, 리더십
   - 프로덕트 매니저 → 비즈니스, 전략, IT, 프로젝트
   - 데이터 사이언티스트 → 데이터, AI, IT, 분석
   - 신사업 기획자 → 비즈니스, 전략, 마케팅
   - 테크 리드 → IT, 개발, 리더십, 기술

3. coursesData에서 필터링
   - 카테고리 매칭
   - 키워드 매칭 (제목, 적용 산업군)

4. 평점 순 정렬 → 상위 5개 선정

5. 우선순위 부여
   - 🔥 최우선: 1순위 (약점 집중 개발)
   - ⭐ 추천: 2순위 (심화 학습)
   - 3. 4. 5.: 3-5순위 (실전 적용)
```

function generateCourseRecommendation(userMessage) {
    // 1. coursesData 로드 확인
    if (typeof coursesData === 'undefined' || coursesData.length === 0) {
        return fallback_message;
    }
    
    // 2. KAPP 점수에서 약점 분석
    const weakestArea = findWeakestArea(userScores);
    
    // 3. 카테고리 매핑
    const targetCategories = categoryMap[weakestArea];
    
    // 4. 강의 필터링
    let recommendedCourses = coursesData.filter(course => {
        return targetCategories.some(cat => 
            course.category && course.category.includes(cat)
        );
    });
    
    // 5. 목표 직무별 추가 필터링
    if (goal) {
        const keywords = goalKeywords[selectedGoal];
        recommendedCourses = recommendedCourses.filter(...);
    }
    
    // 6. 평점 순 정렬 + 상위 5개
    recommendedCourses.sort((a, b) => b.rating - a.rating);
    recommendedCourses = recommendedCourses.slice(0, 5);
    
    // 7. 응답 생성

}
