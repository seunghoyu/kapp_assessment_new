**10개 산업 × 3난이도 × 2문제 = 60문제**

### 2️⃣ Application 시나리오 문항 (20문제)
**10개 산업 × 2문제 = 20문제**

#### 특징
- 실무 상황 기반 시나리오
- 예외 상황 대응 능력 측정
- 데이터 기반 의사결정 평가

#### 예시: IT Backend 개발자
```
상황: 프로덕션 API 응답시간 200ms → 3초 급증
- DB 쿼리: 50ms (정상)
- 외부 API: 2.8초 (급증) ⚠️
- CPU: 30% (정상)

고객 불만 증가 중, 팀장 회의 중. 어떻게 대응?

정답: 외부 API 캐싱 + 타임아웃 1초 설정
→ 즉각적인 완화 조치 + 서비스 안정성 유지
```

### 3️⃣ Performance KPI 문항 (20문제)
**10개 산업 × 2문제 = 20문제**

#### 특징
- 명확한 KPI 제시
- 비용-효율성 판단
- 목표 달성 전략 선택

#### 예시: IT Backend 개발자
```
분기 목표:
- API 응답 시간 ≤ 200ms
- 트래픽 +50% 증가
- 현재: 180ms, 100만 req/일, 월 500만원

최적화 전략?
A. 서버 스케일 업 (성능 2배, 비용 2배)
B. Redis 캐싱 + CDN ✅ (읽기 -70%, 비용 +20%)
C. DB 인덱스만
D. 목표 하향 조정

정답 B 결과:
- 응답시간: 120ms ✅
- 트래픽: 150만 (+50%) ✅
- 비용: 월 600만원 (↑20%)
```

**파일**: `js/kapp-performance.js`

### 4️⃣ Productivity E-tray 시뮬레이션 (10개)
**10개 산업별 이메일 관리 시뮬레이션**

#### 특징
- 제한 시간: 10분
- 이메일 5개 (긴급/중요/일반/참고)
- 액션: reply, forward, delete, later
- 우선순위 판단 능력 측정

#### 예시: IT Backend 개발자
```
📧 이메일 1 (🔴 긴급 - 30점)
From: CTO
Subject: 프로덕션 API 장애 발생
→ 즉시 답장 (reply)

📧 이메일 2 (🟡 일반 - 15점)
From: 팀장
Subject: 코드 리뷰 요청 (내일 배포)
→ 나중에 처리 (later)

📧 이메일 3 (🟢 낮음 - 10점)
From: 신입 개발자
Subject: Docker 설정 질문
→ 멘토에게 전달 (forward)
```

### 5️⃣ AI 워크플로우 시뮬레이션 (10문제) 🆕
**10개 산업별 AI 도구 활용 능력 측정 (마지막 문항)**

#### 🎯 핵심 개념: "업무 가속도"
- 투입 시간 대비 산출물의 품질 측정
- AI 도구 선택 및 워크플로우 설계 능력
- 시간 절감율, 품질 점수, 자동화 레벨 평가

#### 예시 1: IT - AI 코드 리뷰 자동화
```
과제: 일일 PR 15개, 리뷰 시간 7.5시간
목표: 리뷰 시간 50% 단축 + 품질 유지

AI 도구 선택지:
A. GitHub Copilot (코드 자동 완성)
B. SonarQube + AI 리뷰 봇 ✅
C. ChatGPT 수동 질의
D. AI 없이 수동 유지

정답 B 워크플로우:
1. PR 생성 → SonarQube 자동 정적 분석
2. AI 봇이 코딩 컨벤션/보안/성능 자동 검출
3. 사람은 비즈니스 로직만 집중 리뷰
4. AI 요약 검토 후 승인

결과:
- 시간 절감: 60% (7.5h → 3h)
- 품질 점수: 95점
- 자동화 레벨: High
```

#### 예시 2: 금융 - AI 고객 세그먼트 분석
```
과제: 월 50만 고객 데이터 분석, 주 20시간 소요
목표: 분석 시간 70% 단축 + 정확도 향상

정답: Python + AutoML 플랫폼 (H2O.ai)

워크플로우:
1. SQL → Python 자동 데이터 로드
2. AutoML이 최적 세그먼트 자동 클러스터링
3. AI가 각 세그먼트 특성 자동 분석
4. 마케팅 전략 초안을 AI가 작성
5. 사람은 전략 검토 및 최종 승인

결과:
- 시간 절감: 75% (20h → 5h)
- 정확도: 92점
- 자동화 레벨: High
```
