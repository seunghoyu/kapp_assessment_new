# 📈 연봉 시뮬레이터 시계열 그래프 고도화 v2.6

## 🎯 개요
사용자 요청에 따라 연봉 시뮬레이터에 **3년 후 시장 가치 예측 그래프**를 추가했습니다.  
스킬 강화 시 vs 현재 스킬 유지 시 연봉 변화를 시각적으로 비교할 수 있습니다.

---

## ✨ 주요 기능

### 1. 시계열 그래프 (3년 예측)
- **X축**: 현재, 1년 후, 2년 후, 3년 후
- **Y축**: 연봉 (자동 범위 조정)
- **파란 실선**: Upskilling 후 예상 연봉 (선택한 스킬 강화)
  - 그라데이션 효과로 증가분 강조
  - 3px 두께의 선명한 라인
- **회색 점선**: 현재 스킬 유지 시 예상 연봉
  - 5px 간격의 점선 스타일
  - 시장 평균 성장률 반영 (연 3%)

### 2. 실시간 성장률 배지
- 스킬 선택 시 **자동으로 성장률 계산**
- 우측 상단에 `+12%` 형태의 배지 표시
- 녹색 그라데이션 + 펄스 애니메이션
- 스킬 미선택 시 자동 숨김

### 3. 인터랙티브 업데이트
- 스킬 선택 시 즉시 그래프 갱신
- Y축 범위 자동 조정 (최적 시각화)
- 툴팁 호버로 정확한 수치 확인 가능

## 📊 데이터 계산 로직

### 연봉 예측 알고리즘

#### 현재 스킬 유지 시 (회색 점선)
```javascript
현재: baseSalary
1년 후: baseSalary + (baseSalary × 0.03)  // +3%
2년 후: baseSalary + (baseSalary × 0.06)  // +6%
3년 후: baseSalary + (baseSalary × 0.09)  // +9%
```

#### Upskilling 후 (파란 실선)
```javascript
// totalIncrease = 선택한 스킬들의 연봉 증가 합계

현재: baseSalary
1년 후: baseSalary + (totalIncrease × 0.4) + (baseSalary × 0.03)  // 40% 스킬 + 시장
2년 후: baseSalary + (totalIncrease × 0.7) + (baseSalary × 0.06)  // 70% 스킬 + 시장
3년 후: baseSalary + totalIncrease + (baseSalary × 0.09)          // 100% 스킬 + 시장
```

### 성장률 계산
```javascript
3년 차 차이 = upskillingData[3] - currentSkillData[3]
성장률(%) = Math.round((3년 차 차이 / currentSkillData[3]) × 100)
```

## 🧪 테스트 방법

### 1. 기본 그래프 확인
1. `dashboard-kapp.html` 열기
2. "📈 3년 후 시장 가치 예측 그래프" 섹션 찾기
3. 초기 상태: 두 선이 겹쳐 있음 (스킬 미선택)
4. 성장률 배지: 숨김 상태

### 2. 스킬 선택 후 그래프 업데이트
1. "💼 강화할 핵심 스킬 선택" 섹션에서 스킬 선택
   - 예: Python 선택
2. 목표 레벨 클릭
   - 예: Lv.5 클릭
3. **즉시 확인**:
   - ✅ 파란 선이 위로 이동
   - ✅ 우측 상단에 `+12%` 배지 표시
   - ✅ Y축 범위 자동 조정

### 3. 다중 스킬 테스트
1. 여러 스킬 추가 (Python, 데이터 분석, 머신러닝)
2. 각 스킬마다 목표 레벨 설정
3. **기대 결과**:
   - 파란 선이 더 가파르게 상승
   - 성장률 배지: `+18%` → `+25%` 등 증가
   - 그래프가 부드럽게 업데이트 (애니메이션 효과)

### 4. 호버 툴팁 확인
1. 그래프 위에 마우스 올리기
2. "Upskilling 후 예상 연봉: 6,800만원" 형태로 표시
3. "현재 스킬 유지 시: 5,900만원" 비교 가능
