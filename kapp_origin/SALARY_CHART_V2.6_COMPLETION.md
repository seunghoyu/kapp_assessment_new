- 회색 점선: 현재 스킬 유지 시
- 우측 상단: `+12%` 성장률 배지

## ✨ 구현 내용

### 1. 시계열 그래프 렌더링 (`js/dashboard-kapp.js`)
```javascript
function renderSalaryGrowthChart() {
    // Chart.js 초기화
    // 3단계 그라데이션 배경 적용
    // 파란 실선 + 회색 점선 데이터셋
    // 반응형 Y축 범위 설정
}
```

**주요 특징**:
- ✅ **그라데이션**: `rgba(102, 126, 234, 0.25)` → `0.12` → `0.02`
- ✅ **점선 스타일**: `borderDash: [5, 5]`
- ✅ **부드러운 곡선**: `tension: 0.4`
- ✅ **툴팁 포맷**: "6,800만원" 형태로 표시

### 2. 실시간 업데이트 로직 (`js/dashboard-kapp.js`)
```javascript
function updateSalaryGrowthChart() {
    // 선택된 스킬의 totalIncrease 계산
    // Upskilling 트렌드 라인 생성 (40% → 70% → 100%)
    // 현재 스킬 유지 라인 생성 (연 3% 성장)
    // Y축 범위 동적 조정
    // 성장률 배지 업데이트
}
```

**계산 공식**:
```
Upskilling:
- 1년: base + (스킬 × 0.4) + (base × 0.03)
- 2년: base + (스킬 × 0.7) + (base × 0.06)
- 3년: base + (스킬 × 1.0) + (base × 0.09)

성장률: ((3년 Upskilling - 3년 현재) / 3년 현재) × 100
```
