# 해커스 캠퍼스 Business NEXT GEN – 기획자용 프로젝트 개요

> **역할**: 제품/서비스 기획, 사용자 여정, 기능 범위, 데이터 흐름 이해  
> **초점**: 무엇을, 누구를 위해, 어떤 순서로 제공하는지

---

## 1. 프로젝트 비전과 목표

### 1.1 한 줄 정리
**AI 기반 KAPP 역량 진단으로 임직원의 성장을 측정·증명하고, 진단 결과에 맞춘 교육 큐레이션과 관리자용 조직 역량 인사이트를 제공하는 원스톱 HR 솔루션**

### 1.2 핵심 가치
- **진단**: 15분 내 KAPP 4차원(Knowledge, Application, Performance, Productivity) 측정, 8가지 인터랙티브 문항(시나리오·시뮬레이션·E-tray·AI 워크플로우 등)
- **성장 증명**: 대시보드에서 나의 위치, 직급 평균 비교, 스킬 성장 추이, 연봉 예측, 역량 인증서(LinkedIn 공유)
- **교육 연계**: 진단 기반 맞춤 추천 교육, 학습 경로, IDP(개인 개발 계획)
- **조직 인사이트**: 관리자용 스킬 갭 히트맵, 업계 벤치마킹 레이더, 전략–스킬 매핑, 교육비 ROI, 고성과자 분석, 스킬 리스크

### 1.3 사용자 역할
| 역할 | 대상 | 주로 쓰는 화면 |
|------|------|----------------|
| **일반 사용자(임직원)** | 진단 응시자, 학습자 | 홈, KAPP 진단, 마이 대시보드, 교육 큐레이션, 나의 성장, (알림·설정) |
| **관리자(교육담당·HR)** | 조직 단위 현황·계획 | 관리자 대시보드(통계, 부서 차트, 진행 현황, 임직원 테이블, 스킬 갭, 벤치마킹, 전략 매핑, ROI, 고성과자, 리스크) |

## 2. 사용자 여정과 데이터 흐름

### 2.1 전체 플로우
1. **진입**: 홈(랜딩) → KAPP 진단 시작
2. **정보 입력**: 이름, 산업군, 직무, 직급, 연차, 회사 규모, 진단 목표(선택) 등
3. **진단 진행**: 적응형 문항(K-A-P-P)·시나리오·E-tray·AI 워크플로우 시뮬레이션 등 단계별 진행, 진행률 표시
4. **결과 저장**: 진단 완료 시 결과를 브라우저 저장소에 저장 후 **마이 대시보드(KAPP 대시보드)** 로 자동 이동
5. **대시보드 소비**: 저장된 결과로 DNA 유형, KAPP 점수, 레이더, 시장 포지션, 커리어 경로, 벤치마킹, 성장 추이, AI 인사이트, IDP, 게이미피케이션 등 표시
6. **교육 탐색**: 교육 큐레이션에서 진단 기반 추천 + 필터(산업·역량·난이도·기간)로 전체 과정 탐색, 과정 상세 모달
7. **지속 참여**: 나의 성장에서 오늘의 1% 팁, AI 커리어 멘토링, 역량 인증서
8. **관리자**: 관리자 페이지에서 조직 통계, 부서별 차트, 진행 현황, 임직원 테이블, 스킬 갭·벤치마킹·전략 매핑·ROI·고성과자·리스크 확인 및 액션

### 2.2 데이터가 쓰이는 방식 (현재 구현 기준)
- **진단 결과**: 진단 완료 시 한 번 저장. 대시보드·교육 추천·나의 성장·IDP·게이미피케이션 등이 이 데이터를 읽어서 표시·추천
- **사용자 정보**: 진단 시 입력한 이름·산업·직무·직급·연차 등이 결과와 함께 저장되어 대시보드·커리어 경로·시장 포지션 등에 반영
- **관리자 데이터**: 부서별 점수, 임직원 목록, 부서별 스킬, 업계 벤치마크, 전략 키워드–스킬 매핑, 고성과자·리스크 등은 모두 **프론트에서 정의한 더미/목 데이터**로 동작 (추후 API 연동 가정)

## 3. 페이지별 기능과 목적

### 3.1 홈(랜딩)
- **목적**: 서비스 소개, 진단 유도
- **구성**: 히어로, 핵심 수치(15분 진단·97% 정확도·91% 완료율 등), 왜 해커스인가(게이미피케이션·KAPP·실무 평가·AI 워크플로우), 기능 하이라이트, CTA(진단 시작·둘러보기)
- **로직**: 클릭 시 KAPP 진단 페이지 또는 앵커로 이동

### 3.2 KAPP 진단
- **목적**: 사용자 정보 수집 + KAPP 4차원 역량 측정
- **단계**:
  1. **정보 입력**: 기본 정보(이름·이메일), 산업·직무(산업 선택 시 직무 목록 연동), 직급·연차, 회사·기업 규모, 진단 목표(다중 선택)
  2. **진단 문항**: 카테고리별(K-A-P-P)·난이도별 적응형 문항, 산업별 문항 풀, 시나리오/선다형/E-tray(인바스켓)/AI 워크플로우 시뮬레이션 등
  3. **진행률**: 퍼센트·현재 단계명 표시
- **완료 시**: 결과 객체(사용자 정보 + KAPP 점수 등) 저장 후 **마이 대시보드(dashboard-kapp)** 로 리다이렉트

### 3.3 마이 대시보드(KAPP 대시보드)
- **목적**: 진단 결과를 한 화면에서 요약·시각화하고, 성장·교육·공유로 이어지게 함
- **데이터 소스**: 저장된 진단 결과(없으면 샘플 데이터로 표시)
- **주요 블록**:
  - **나의 업무 DNA**: KAPP 점수 기반 DNA 유형·설명·트레이트, LinkedIn 공유·배지 다운로드
  - **KAPP 4차원 점수**: K·A·P·P 점수 카드 + 레이더 차트(나 vs 직급 평균)
  - **시장 포지션 분석**: 현재 포지션·백분위·종합 점수, 경쟁력(강점/개선/평균 스킬 수), 시장 수요도, 성장 기회 타임라인, 산업군 벤치마크 레이더(상위 10% vs 나), 추천 액션, 맞춤 성장 플랜 CTA
  - **커리어 경로 시뮬레이터**: 산업·직무 기반 3~5년 경로 타임라인, 추천 학습 경로, 추천 강의 CTA
  - **시장 벤치마킹**: 동일 직급 평균 vs 나 vs 상위 백분위, 막대 차트
  - **스킬 성장 추이**: 최근 6개월 추이 라인 차트
  - **AI 분석 인사이트**: 점수·갭 기반 텍스트 인사이트 카드
  - **AI 생성 IDP**: 스킬 갭 기반 개인 개발 계획(목표·기간·추천 교육)
  - **게이미피케이션**: 레벨·경험치·배지·연속 학습 등
  - **3년 연봉 예측**: 시계열 그래프(산업·직무 기반)
- **액션**: 재진단, 리포트 다운로드(인쇄), LinkedIn 공유, 배지 다운로드, 성장 플랜·강의 페이지 이동

### 3.4 교육 큐레이션
- **목적**: 진단 기반 맞춤 추천 + 전체 과정 탐색
- **데이터**: 과정 목록(산업·카테고리·난이도·기간·추천 여부 등 속성 포함)
- **로직**:
  - **맞춤 추천**: 저장된 진단 결과가 있으면 그에 맞춰 추천 과정 노출, 추천 사유 문구 표시
  - **필터**: 산업군, 역량 분야, 난이도, 학습 기간 선택 시 목록 필터링
  - **전체 과정**: 그리드/리스트 뷰 전환, 과정 카드 클릭 시 **상세 모달**(제목·카테고리·난이도·설명·커리큘럼·학습 결과·수강 신청 등)
  - **추천 학습 경로**: 역량별(문제 해결·의사소통·리더십·학습·기술·협업 등) 단계별 경로 + 연결된 과정

### 3.5 나의 성장
- **목적**: 일상 참여(1% 팁) + AI 멘토링 + 역량 인증
- **오늘의 1% 효율 챌린지**:
  - 매일 1분 팁 카드(생산성·커뮤니케이션·AI·엑셀 등 카테고리)
  - 새 팁 받기, 적용 완료, 나중에 보기
  - 연속 학습 일수·완료 팁 수·주간 목표(0/7) 표시
  - 지난 팁 다시보기 목록
  - 진행 상황은 브라우저 저장소에 유지
- **AI 커리어 멘토링**:
  - 선망 커리어 선택(테크 리드·데이터 사이언티스트·금융 애널리스트·의료 데이터 관리자 등)
  - 채팅 형태 질문 입력 → AI 응답(교육 추천·기간·전략·연봉 전망·동기부여 등 시나리오별 답변)
- **역량 인증서**:
  - 진단 결과 기반 인증서 미리보기, LinkedIn 공유·다운로드

### 3.6 알림·설정
- **목적**: 푸시 알림 권한 요청, 알림 항목 on/off, 학습 진행 설정
- **로직**: 오늘의 팁 알림·연속 학습 알림·학습 경로 추천 등 토글 시 설정 저장, 푸시 허용 시 저장

### 3.7 관리자 대시보드
- **목적**: 조직 단위 역량·교육 현황과 인사이트로 의사결정 지원
- **데이터**: 전부 프론트 정의 데이터(부서·임직원·스킬·업계 벤치마크·전략 키워드 등)
- **주요 블록**:
  - **요약 카드**: 전체 임직원 수, 진단 완료 수/완료율, 교육 이수 건수, 평균 역량 점수
  - **부서별 역량 현황**: 기간 선택(월별) → 해당 월 부서별 평균 점수 막대 차트
  - **역량 분포**: 파이/도넛 등 분포 차트
  - **역량 개발 진행 현황**: 카드 목록(역량명·참여 인원·진행률·평균 점수·목표), 필터(전체/진행중/완료)
  - **임직원 역량 현황**: 테이블(이름·부서·직급·종합 점수·강점·개선 필요·교육 현황·액션), 검색·임직원 추가 버튼
  - **AI 인사이트**: 긴급 조치·개선 기회·목표 달성 등 텍스트 카드
  - **스킬 갭 히트맵**: 부서·스킬 카테고리 선택 → 부서별 스킬 셀 컬러(긴급/주의/보통/양호/우수), 교육 배정 버튼
  - **업계 벤치마킹 레이더**: 비교 부서·비교 업계 선택 → 레이더(우리 부서 vs 업계 상위 10%), 인사이트(개선 필요 영역·경쟁 우위), 개선 계획 수립 버튼
  - **조직 전략–스킬 매핑**: 전략 키워드 입력 → AI 분석 → 추천 인력 배치·필수 교육·실행 로드맵, 인력 배정·일정 등록 버튼
  - **교육비 ROI 계산기**: 대상 인원·1인당 비용·예상 스킬 향상 % 입력 → 총 투자·예상 수익·순이익·ROI %, 상세 분석·보고서 다운로드
  - **고성과자 복제 모델링**: 팀 선택 → 상위 10% Golden Standard 레이더, 갭 분석
  - **스킬 리스크 관리**: 리스크 레벨 필터 → 긴급/높음/보통/낮음 카운트, 상세 리스트

## 4. 주요 로직 요약(기획 관점)

### 4.1 진단 → 대시보드
- 진단 제출 시 **결과 JSON**을 브라우저 저장소에 저장하고, **마이 대시보드**로 이동
- 대시보드는 저장된 결과를 읽어 DNA·점수·레이더·시장 포지션·커리어·IDP 등 모든 섹션을 채움. 없으면 샘플로 표시

### 4.2 추천·필터
- **교육**: 진단 결과(산업·직무·점수)와 과정 속성 매칭으로 추천, 나머지는 산업·역량·난이도·기간 필터로 제한
- **관리자**: 부서·기간·업계·팀·리스크 레벨 등 선택 시 해당 데이터만 사용해 차트·테이블·인사이트 갱신

### 4.3 모달·알림
- **교육 과정 상세**: 카드 클릭 → 모달 열림(제목·설명·커리큘럼·수강 신청 등), 닫기로 종료
- **개선 계획 수립(벤치마킹)**: 스킬별 버튼 클릭 시 안내 메시지(목표·기간·추천 교육·담당자 알림)
- **전략 매핑·ROI·고성과자·인력 배정·일정 등록**: 버튼 클릭 시 안내(토스트/알림)로 완료 처리

### 4.4 게이미피케이션·성장
- 연속 학습 일수·완료 팁 수·주간 목표는 **나의 성장**에서 로컬 저장
- 대시보드 게이미피케이션은 진단 결과·완료 여부 기반 레벨·XP·배지 계산(로컬 저장)

## 5. 비기능 요구사항(기획 관점)

- **반응형**: 주요 화면 모바일·태블릿 대응
- **접근성**: 주요 버튼·폼·차트에 의미 있는 라벨·구조 유지
- **성능**: 차트·대량 목록은 필요 시 지연 로딩·캔버스 재사용으로 부드러운 전환
- **확장**: 산업·직무·문항·과정·전략 키워드는 데이터만 추가하면 확장 가능하도록 설계

## 6. 정리
